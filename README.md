# NuclearEnergy_Industrial_decarbonization-PyPSA-
Exploring the use of Small Modular Reactors for decarbonizing industries in Spain. 
The code is in de jupyter notebook called SMR_multilink.ipynb. 


## Technical tips: 
### 1) Shared Environment.yml 
There is an archive called PYsimul2env.yml that contains all the dependencies and exact versions of the original environment configuration in which the code is being written (in VS Code with Anaconda). To use it try typing this in the anaconda terminal after downloading it:

_conda env create -f PYsimul2env.yml_ (creates a new environment based in "PYsimul2env.yml")

_conda activate PYsimul2env (activates de new environment called PYsimul2env)
